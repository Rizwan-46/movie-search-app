# 🎬 Movie Search App

A simple React application that lets users search for movies and view their details using a movie API.

## ✨ Features

* Search movies by title
* View movie details
* Responsive design
* Fast and easy-to-use interface

## 🛠️ Built With

* React
* CSS
* Movie API (OMDb or TMDb)
* npm


## 🚀 Getting Started

### Prerequisites

* Node.js
* npm

### Installation

Clone the repository:

```bash
git clone https://github.com/YourUsername/movie-search-app.git
```

Go to the project folder:

```bash
cd movie-search-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

**For Vite**

```bash
npm run dev
```

**For Create React App**

```bash
npm start
```

## 📦 Build for Production

```bash
npm run build
```
